# Bài tập Git-GitHub

Đây là repo demo cho bài tập 1 về Git và GitHub.

## Nội dung bài tập
- Tạo repo trên local
- Phân nhánh
- Tạo repo trên github và kết nối repo giữa local và remote github
- Push kết quả lên github
- Clone từ github về local
- Merge các nhánh trong git về main
- Giải quyết xung đột nếu có

## Deploy dự án bằng GitHub Actions (CI/CD)
- Chọn các ví dụ mẫu nhỏ cho các project sau (có frontend - backend - database)
